Instructions

  1. Click "Initialize StyloEditor" to start the editor normally
  2. Click "Initialize Minimized" to start the editor in minimized mode
  3. The editor panel will appear on the screen
  4. Click on any element to select and edit it
  5. Use the panel to modify styles in real-time
  6. Click "Toggle Inspector" to enable/disable element selection
